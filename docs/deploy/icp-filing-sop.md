# ICP 备案完整 SOP（阿里云）

> 给 voice agent 服务的**自有域名**办 ICP，让国内合法访问。
> 预计时间：**7-20 天**（个人备案通常 7-10，企业 10-20）。
> 在你等备案的 1-3 周里，**take-home 该交就交、用 WebRTC 演示版**——两件事不冲突。

---

## 0. 备案前准备清单

- [ ] 一个**未备案**的 `.cn` / `.com.cn` / `.cn` 域名（最便宜 30 元/年）
  - 推荐注册商：阿里云万网 / 腾讯云 / 新网
  - 建议：`voice.你的公司域名.cn` 或 `agent.你的域名.com`
- [ ] 身份证（个人）/ 营业执照（企业）+ 法人身份证
- [ ] 手机号（实名认证过的，跟身份证一致）
- [ ] 阿里云账号（实名认证完成）
- [ ] 阿里云 ECS 至少买 1 个月（不买 ECS 也能备案，但要填"待购买"）

## 1. 域名注册（如果还没有）

1. https://wanwang.aliyun.com → 搜你想要的域名
2. 注册 → **必须完成实名认证**（否则不能备案）
3. 实名认证 1-2 个工作日审核

## 2. 阿里云账号实名

1. https://account.aliyun.com → 账号管理 → 实名认证
2. 个人 / 企业二选一
3. 上传身份证 / 营业执照
4. 1-2 个工作日通过

## 3. 购买 ECS（最小配置）

备案**要求服务器在阿里云**（或你选的云）。先买 1 个月跑起来。

| 配置 | 推荐 |
|---|---|
| 区域 | **华北 2（北京）**或**华东 1（杭州）**（离 Bailian API 近） |
| 规格 | 1 vCPU / 2 GB 内存 / 40 GB SSD 足够 |
| 镜像 | Ubuntu 22.04 LTS 64位 |
| 带宽 | 1-5 Mbps（按量付费） |
| 价格 | ~¥50-100/月 |

## 4. 进入备案系统

1. 登录 https://beian.aliyun.com
2. **开始备案** → 填写主办者信息（你或你公司）
3. **填写网站信息**：
   - 网站名称：取个能过的，比如"蓝色鲸鱼访客登记系统"
   - 网站域名：`voice.你的域名.cn`
   - 服务内容：选"其他"或"信息技术"
4. **上传资料**：
   - 身份证正反面
   - 域名证书（在域名管理后台下载）
   - 网站备案真实性核验单（阿里云自动生成，下载打印，手写签名后拍照上传）

## 5. 阿里云初审（1-3 个工作日）

- 阿里云客服会打电话给你确认
- 答得快一般 1 天过
- 不通过会告诉你哪里要改

## 6. 管局审核（7-20 个工作日）

- 工信部审核
- 期间你的域名**解析到阿里云 ECS**才能通过（所以 ECS 先开起来）
- 可以在 https://beian.aliyun.com 查进度

## 7. 备案成功后

工信部会给你一个**备案号**（`京ICP备XXXXXXXX号`）。把这个号**放在你网站底部**（法律规定）。

---

## 备案期间域名能干嘛？

- ❌ 不能**解析到中国大陆服务器**（会阻断）
- ✅ 可以**解析到海外服务器**（你现在 cloudflared 跑美国，就能用！）
- ✅ 可以**只用 IP 访问**（不解析域名）

> **所以 take-home 期间你保持 cloudflared 跑就行**，备案过了再切到阿里云。

---

## 常见被打回的原因

| 原因 | 怎么避免 |
|---|---|
| 域名没实名 | 注册后立刻实名 |
| 网站名称不规范 | 别带"中国""国家""最"等敏感词 |
| 身份证与域名实名不一致 | 域名实名用谁就备案用谁 |
| 手机号非本人 | 换成本人身份证办的号 |
| 主体负责人未到现场拍照 | 部分省份要"当面核验"，看阿里云通知 |
